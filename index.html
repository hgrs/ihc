Solução 360

Solução de hardware e software que controla tudo.
A solução 360 é um conceito que criamos para trazer o universo da automação residencial mais perto de você com uma visão voltada para a Internet das coisas.
Esta revolução já iniciou e gostaríamos de poder trazer este universo até você através do seu celular controlando as coisas a sua volta de uma forma integrada independentemente de marca, fabricante ou tipo de equipamento.

Núcleos

Nosso objetivo, primeiramente, é permitir que você conheça este universo chamado automação residencial e internet das coisas, e consiga trazer para o seu dia-a-dia esta realidade de forma prática e usual. Isto se faz através de um software e um hardware universais que permite conectar com todas as marcas e equipamentos de forma isolada ou simultânea.

Simples

Nenhuma programação. Com a Solução IHC 360 On Controls você apenas arrastar e soltar imagens do banco de dados para criar e modificar a interface remota.

Flexível

Use os sistemas de sua preferência e personalize a interface do usuário com as necessidades e desejos dos clientes com as necessidades e desejos dos clientes.

Remoto

Construir projetos de controle na nuvem é ter acesso e modificar remotamente o projeto de um cliente sem ter que se dirigir para casa ou empresa dos clientes.

Bonito

Use a biblioteca incluída de centenas de imagens para personalizar a interface sem gastar dinheiro em uma biblioteca de gráficos.
